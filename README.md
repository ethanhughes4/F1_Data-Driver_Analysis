# 🏎️ Formula 1 Analytics Dashboard

An end-to-end data analytics project analysing historical Formula 1 race data using **PostgreSQL, SQL, Power BI, Power Query and DAX**. The project demonstrates the complete analytics workflow from data preparation and SQL analysis to building an interactive Power BI dashboard.

## 📖 Project Overview

This project analyses historical Formula 1 race data to uncover trends in driver, constructor and race performance. The dataset was imported into PostgreSQL, analysed using SQL, transformed in Power Query and visualised in an interactive Power BI dashboard.

## ⚙️ Project Workflow

### SQL

- Imported and structured the Formula 1 dataset in PostgreSQL.
- Wrote analytical SQL queries using joins, aggregates, CTEs, subqueries and window functions.
- Answered business-focused analytical questions.

### Power Query

- Cleaned and transformed the data.
- Corrected data types.
- Created calculated columns.
- Prepared the data for reporting.

### Power BI

- Built a relational data model.
- Created DAX measures for key performance indicators.
- Designed an interactive dashboard.
- Implemented drill-through navigation for detailed driver analysis.

---

## 💡 Key Insights

- Lewis Hamilton has the highest number of career race wins in the dataset.
- Ferrari is one of the most successful constructors in Formula 1 history.
- The Formula 1 calendar has expanded significantly over time, with modern seasons containing more races than earlier decades.
- Interactive filtering allows users to explore driver and constructor performance.
